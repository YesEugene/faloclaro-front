# Как прочитать комментарии донатов в Stripe

## Где хранятся комментарии

Комментарии пользователей сохраняются в Stripe в двух местах:

1. **В описании продукта** (`product_data.description`) - видно в списке платежей
2. **В метаданных сессии** (`metadata.comment`) - доступно в деталях checkout сессии

## Как прочитать комментарии

### Способ 1: Через Payments (Платежи)

1. Откройте Stripe Dashboard: https://dashboard.stripe.com
2. Перейдите в раздел **Payments** (Платежи) в левом меню
3. Найдите нужный платеж (можно фильтровать по дате, сумме и т.д.)
4. Нажмите на платеж, чтобы открыть детали
5. В разделе **Product** вы увидите описание, которое содержит комментарий пользователя (если он был указан)

### Способ 2: Через Checkout Sessions (Сессии оплаты)

1. Откройте Stripe Dashboard: https://dashboard.stripe.com
2. Перейдите в раздел **Products** → **Checkout** → **Sessions**
3. Найдите нужную сессию (можно фильтровать по дате, статусу и т.д.)
4. Нажмите на сессию, чтобы открыть детали
5. Прокрутите вниз до раздела **Metadata** (Метаданные)
6. Там вы найдете:
   - `comment` - комментарий пользователя
   - `email` - email пользователя (если был указан)

## Экспорт данных

Если нужно получить все комментарии сразу:

1. В Stripe Dashboard перейдите в **Payments**
2. Нажмите на кнопку **Export** (Экспорт) в правом верхнем углу
3. Выберите формат (CSV или Excel)
4. В экспортированном файле будет колонка с описанием продукта, где содержатся комментарии

## Важно

- Комментарии сохраняются только если пользователь их указал
- Если комментарий не был указан, в описании будет стандартный текст: "Thank you for supporting FaloClaro!"
- Комментарии доступны только в боевом режиме (Live mode), не в тестовом (Test mode)
