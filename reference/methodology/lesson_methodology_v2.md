## Методология создания урока (v2) — FaloClaro

### Шаг 1. Сначала формируем целевое высказывание урока (результат)

Каждый урок проектируется **от результата к шагам**.
Сначала мы определяем 2–3 связанные фразы, которые ученик должен уметь произнести к концу урока.

Пример целевого результата (урок «Где я и откуда»):

- Sou de Portugal.
- Moro em Lisboa agora.
- É um bom lugar.

### Шаг 2. TASK 2 — ядро урока (строим мысль, а не объясняем грамматику)

TASK 2 не «объясняет правила». Он ведёт ученика от минимальной мысли к целевому сообщению через постепенное наращивание смысла.

**Логика**:
финальная мысль → разбивается на смысловые блоки → каждый блок становится шагом обучения.

**Структура TASK 2**:
- 5–7 блоков (в платформе чаще всего используем 6)
- каждый блок = один логический шаг
- каждое следующее предложение **должно быть длиннее или смыслово богаче** предыдущего
- блок 5 — reinforcement: новые комбинации, без дословного повторения блоков 1–4
- блок 6 — speak out loud: целевое высказывание (тот самый итог урока)

**Типовая прогрессия**:
- Блок 1 — минимальная мысль (2–3 слова)
- Блок 2 — расширение (добавляем деталь/время/контекст)
- Блок 3 — вторая мысль того же уровня
- Блок 4 — связка мыслей (2 предложения подряд)
- Блок 5 — проверка смысла (новые комбинации тех же элементов)
- Блок 6 — выход в речь (цель урока)

**Подводки (explanation_text)**:
- не «про грамматику», а «зачем этот шаг»
- разговорные, средней длины
- всегда подталкивают проговорить вслух

**Подсказки (hint)**:
- практичные опоры, а не лекция
- помогают сразу применить структуру

### Шаг 3. TASK 3 — Listening (узнавание смысла на слух)

Цель: научить ученика **узнавать смысловую структуру**, а не переводить по словам.

Правила:
- нельзя повторять фразы из TASK 2 дословно
- используем те же конструкции, но в **новых комбинациях**
- внутренняя прогрессия: item 1 проще → item 2 связнее → item 3 богаче/длиннее
- 3 варианта ответа, 1 правильный

### Шаг 4. TASK 4 — Attention (быстрое распознавание + подготовка к речи)

Цель: закрепить восприятие целых фраз на слух и ускорить реакцию.

Правила:
- не вводим новые слова
- не повторяем дословно фразы из TASK 2/3
- вопросы про ситуацию/тип высказывания/структуру мысли
- 3–4 варианта ответа
- feedback обязателен: короткий, практичный, направляет проговорить вслух

### Шаг 5. TASK 5 — Writing/Speak (финальный результат)

Цель: перевести понимание в активную речь.

TASK 5 содержит:
- Instruction (RU/EN): объясняет зачем, мотивирует говорить; включает блок:
  - «Для задания используй слова: …» (список PT слов из словаря урока)
- Main assignment (template): структура целевой мысли с пропусками
- Example: скрыт по кнопке, показывает полностью заполненный вариант
- Alternative: «если не хочется писать — скажи вслух полностью без пауз»

### Шаг 6. TASK 1 — словарь из всех заданий

TASK 1 должен включать **все слова**, которые встречаются в Tasks 2–5.
Минимум 13 карточек (10 основных + 3 поддерживающих).

### Важный технический момент (платформа)

- В вариантах ответов используем поле **`correct: true/false`** (а не `is_correct`).
- В аудио-полях всегда должна быть **реальная PT фраза**, никогда не плейсхолдер вроде `"PT"`.
